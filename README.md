# Marvel Reading Vault v3

A local, offline-first Marvel comics roadmap and progress tracker.

## Open the website

1. Extract the ZIP archive.
2. Open `index.html` in a modern browser.
3. Your reading progress is saved in the browser using IndexedDB, with a local fallback.

No installation, account, server, framework, or internet connection is required.

## Main sections

- Dashboard
- Master Flow
- Phases
- Character and team roadmaps
- Structured events and soft prerequisite gates
- Great Stories
- Ultimate Universe
- Elseworlds and crossovers
- Statistics
- Settings and backups

## Tracking model

Every comic issue has one canonical ID. If an issue appears in multiple roadmaps or events, marking it read in one place updates it everywhere.

Current and announced comics are separated. Only published issues count toward completion.

## Route modes

- Peak only
- Peak + Essential
- Recommended route
- Full curated route
- Completionist

## Backups

Use Settings to export progress as JSON. The backup includes issue progress, ratings, notes, ownership fields, favourites, wish-list data, and interface settings.

## Data version

Schema 3, catalogue `2026.06.28-v3`.
