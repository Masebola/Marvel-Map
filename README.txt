Miclase Marvel Universe

How to use:
1. Open index.html in your browser.
2. Choose a phase from the left panel.
3. Open a reading block to see every individual issue.
4. Tick individual issues, mark a full block read, or mark/clear a full phase.
5. Your progress saves automatically in localStorage.
6. Use Export Progress before changing browsers/devices, then Import Progress later.

Files:
- index.html
- style.css
- script.js

Notes:
- Issue totals are calculated from the actual issue entries in each block, not from ranges as single items.
- Optional blocks are still trackable, especially the Bonus post-Secret Wars shelf.
